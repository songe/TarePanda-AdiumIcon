# Tarepanda Adium Dock Icons
Adium dock icon set featuring Tarepanda

## How to install
- Clone the repo to download by running `git clone git@github.com:songe/TarePanda-AdiumIcon.git`
- Open TarePanda.AdiumIcon to install
